GIT COMMANDS QUICK GUIDE
========================

1.  Initialize Repository - git init

2.  Check Status  -git status

3.  Add Files to Staging - git add . git add filename.txt

4.  Commit Changes  - git commit -m “Your message”

5.  Connect to GitHub Repository git remote add origin
    https://github.com/username/repository-name.git

6.  Push Code to GitHub - git branch -M main & git push -u origin main

7.  Pull Latest Changes - git pull origin main

8.  Clone Repository git clone
    https://github.com/username/repository-name.git

9.  View Commit History - git log

10. Create New Branch - git branch branch-name

11. Switch Branch - git checkout branch-name

12. Create and Switch Branch - git checkout -b branch-name
